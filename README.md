---

## Visão Geral

Este repositório foi criado para concentrar todos os artefatos exigidos na atividade da disciplina, reunindo em um só lugar:

- análise orientada a objetos de cada questão;
- identificação de classes, atributos e métodos;
- requisitos funcionais e não funcionais;
- diagramas de classes;
- aplicações web em Python com Streamlit;
- documentação complementar da atividade.

A proposta do trabalho é desenvolver as soluções das **11 questões da Lista de Exercícios 01**, mantendo a organização por questão e separando claramente as partes de **análise**, **diagramas** e **aplicação**.
