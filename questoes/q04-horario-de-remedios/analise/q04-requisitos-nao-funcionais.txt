RNF01 - O sistema deve possuir interface simples e intuitiva.
RNF02 - O sistema deve validar os campos obrigatórios.
RNF03 - O sistema deve exibir os horários de forma clara.
RNF04 - O sistema deve permitir manutenção fácil do código.
RNF05 - O sistema deve ser desenvolvido em Python.
RNF06 - O sistema deve utilizar Streamlit para a interface web.
RNF07 - O sistema deve armazenar os dados de forma organizada.
RNF08 - O sistema deve apresentar resposta rápida na reorganização dos horários.
