# Questão 05 — Gastos Diários

## Enunciado resumido

Vera utiliza uma planilha para controlar seus gastos diários.

Para cada gasto, são cadastradas as seguintes informações:
- tipo do gasto
- data do gasto
- valor do gasto
- forma de pagamento

Ao final do mês, Vera precisa listar o total dos gastos mensais, agrupados por tipo de gasto e exibindo o quanto foi gasto em cada forma de pagamento. :contentReference[oaicite:1]{index=1}

---

## Objetivo da questão

Identificar:
- classes
- atributos
- métodos
- relacionamentos

Além disso:
- definir os requisitos funcionais e não funcionais
- representar o diagrama de classes
- desenvolver a aplicação em Python com Streamlit
