RNF01 - O sistema deve possuir interface simples e de fácil utilização.
RNF02 - O sistema deve validar os campos obrigatórios no cadastro do gasto.
RNF03 - O sistema deve aceitar valores monetários com precisão de duas casas decimais.
RNF04 - O sistema deve exibir os relatórios mensais de forma clara e organizada.
RNF05 - O sistema deve permitir manutenção fácil do código.
RNF06 - O sistema deve ser desenvolvido em Python.
RNF07 - O sistema deve utilizar Streamlit para a interface web.
RNF08 - O sistema deve manter consistência nos dados cadastrados.
RNF09 - O sistema deve apresentar desempenho adequado para consultas mensais.
RNF10 - O sistema deve organizar os dados de modo que facilite futuras expansões da aplicação.
