# :shopping_cart: Questão 07 — Lista de Compras

## :pushpin: Sobre a questão

Esta questão trata da modelagem de uma **lista de compras mensal** controlada por Carolina em uma planilha.

Para cada item da lista, são informados:
- nome do produto
- unidade de compra
- quantidade prevista para o mês
- quantidade que efetivamente será comprada
- preço estimado

Ao final, a lista apresenta o **total da compra**.

### Esclarecimentos do enunciado
- a quantidade de compra pode variar de um mês para outro
- não há necessidade de controlar marcas
- mensalmente, Carolina pode analisar o valor pago por produto e atualizar o preço estimado

---

## :dart: Objetivo

Identificar:
- classes
- atributos
- métodos
- relacionamentos

Além disso:
- especificar requisitos funcionais e não funcionais
- representar o diagrama de classes
- desenvolver uma aplicação em **Python com Streamlit**
