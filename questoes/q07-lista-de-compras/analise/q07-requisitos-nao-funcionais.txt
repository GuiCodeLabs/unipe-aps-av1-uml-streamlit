RNF01 - O sistema deve possuir interface simples e intuitiva.
RNF02 - O sistema deve validar os campos obrigatórios no cadastro dos itens.
RNF03 - O sistema deve aceitar valores numéricos com casas decimais para quantidade e preço.
RNF04 - O sistema deve exibir os totais de forma clara e organizada.
RNF05 - O sistema deve manter consistência nos dados cadastrados.
RNF06 - O sistema deve ser desenvolvido em Python.
RNF07 - O sistema deve utilizar Streamlit para a interface web.
RNF08 - O sistema deve permitir fácil manutenção e evolução do código.
RNF09 - O sistema deve apresentar desempenho adequado para o controle mensal da lista.
RNF10 - O sistema deve organizar os dados de modo que facilite futuras expansões da aplicação.
