# Aplicação - Questão 3: Boneco em Movimento

## Descrição

Esta aplicação foi desenvolvida em **Python** utilizando **Streamlit** para representar, de forma prática, a solução da questão 3.

O sistema permite movimentar um boneco na tela, considerando:
- nome
- posição X
- posição Y
- direção atual

A direção do boneco pode ser:
- cima
- baixo
- esquerda
- direita

---

## Funcionalidades

A aplicação permite:

- informar o nome do boneco
- definir a posição inicial em X e Y
- escolher a direção atual
- mover o boneco para cima
- mover o boneco para baixo
- mover o boneco para a esquerda
- mover o boneco para a direita
- visualizar o estado atual do boneco
- reiniciar a posição do boneco
- exibir uma representação simples da posição na tela
