# :bulb: Aplicação — Questão 01

## :pushpin: Sobre a aplicação

Esta aplicação foi desenvolvida em **Python com Streamlit** para atender à Questão 01 — **Conta de Luz**.

O sistema permite cadastrar contas mensais de energia elétrica, visualizar os registros cadastrados e consultar o mês de menor e maior consumo.

---

## :white_check_mark: Funcionalidades

- cadastrar conta de luz
- listar contas cadastradas
- editar conta existente
- excluir conta cadastrada
- identificar menor consumo
- identificar maior consumo
- exibir resumo geral dos registros

---

## :computer: Tecnologias utilizadas

- Python
- Streamlit
- Pandas

---

## :rocket: Como executar

No terminal, dentro da pasta `app`, execute:

```bash
pip install streamlit pandas
streamlit run app.py
