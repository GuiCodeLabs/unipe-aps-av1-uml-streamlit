# Questão 02 — Prompts da aplicação

## Sobre a questão

A questão 02 trata da classe **TextoSaída**, cujo objetivo é configurar um texto por meio de atributos como:
- tamanho da letra
- cor da fonte
- cor do fundo
- tipo de componente de exibição (`label`, `edit`, `memo`)

## Prompt 1 — Geração do app.py

Use a descrição da questão 02 como base para gerar uma aplicação web em Python utilizando Streamlit.

Contexto da questão:
A classe TextoSaída deve permitir configurar um texto por meio dos atributos tamanho da letra, cor da fonte e cor do fundo, escolhendo em que tipo de componente ele deve ser exibido, entre as opções: label, edit e memo.

Requisitos para a aplicação:
- identificar a classe principal e seus atributos e métodos com base no enunciado
- implementar o sistema em Python
- usar Streamlit como interface
- permitir ao usuário digitar um texto
- permitir escolher o tamanho da letra
- permitir escolher a cor da fonte
- permitir escolher a cor do fundo
- permitir escolher o tipo de componente visual entre label, edit e memo
- exibir o texto configurado na interface
- o código deve ser completo e funcional
- gerar tudo em um único arquivo `app.py`
