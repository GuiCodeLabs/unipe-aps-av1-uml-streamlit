# Questão 08 — Prompts da aplicação

## Sobre a questão

A questão 08 trata de uma aplicação para **coleção de CDs**, em que Adriano deseja cadastrar:
- nome do cantor(a) ou conjunto
- título do CD
- ano de lançamento

O objetivo é organizar sua coleção e facilitar a consulta dos CDs cadastrados.

## Prompt 1 — Geração do app.py

Use a descrição da questão 08 como base para gerar uma aplicação web em Python utilizando Streamlit.

Contexto da questão:
Adriano possui uma coleção grande de CDs e deseja cadastrar a lista de CDs em um sistema. Para cada CD, devem ser armazenados o nome do cantor ou conjunto, o título do CD e o ano de lançamento.

Requisitos para a aplicação:
- identificar classes, atributos e métodos com base no enunciado
- implementar o sistema em Python
- usar Streamlit
- permitir cadastrar CDs
- permitir informar cantor ou conjunto
- permitir informar título do CD
- permitir informar ano de lançamento
- listar os CDs cadastrados
- permitir pesquisar por título ou artista
- permitir editar e excluir registros
- o código deve ser funcional
- gerar tudo em `app.py`
